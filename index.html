About me

I am a dedicated Cyber Security Analyst with skills in threat detection, network security, and vulnerability assessment. I have hands-on experience with tools like Wireshark, Nessus, and Splunk, and a solid grasp of security protocols and firewalls.

I follow frameworks like NIST and ISO 27001, and enjoy staying updated on emerging cyber threats. My goal is to help protect organizations with proactive security strategies and fast incident response.

What I'm Doing

  • Threat Monitoring & Detection

    Continuously monitoring systems, networks, and logs to identify suspicious activities or cyber threats using tools like SIEM (e.g., Splunk) and intrusion detection systems.

  • Vulnerability Assessment

    Scanning networks and applications for security weaknesses using tools like Nessus or OpenVAS, and recommending fixes before attackers can exploit them.

  • Incident Response

    Investigating and responding to security incidents, analyzing the root cause, containing threats quickly, and helping recover affected systems.

  • Security Policy & Compliance

    Creating and updating security guidelines, ensuring compliance with standards like ISO 27001 or NIST, and training staff on best security practices.
